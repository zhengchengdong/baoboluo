## 1. 游戏概述

- **进攻方**：点击菠萝表面逐根绑皮筋，皮筋收缩挤压，直到爆裂
- **防守方**：割断皮筋释放压力

---

## 2. PBD 核心

### 2.1 基本概念

PBD 不显式计算力。定义**约束**，每帧反复修正顶点位置来满足它们。每条约束的修正公式：

```
Δp = (目标值 − 当前值) × 刚度
```

刚度 < 1，每次只朝目标挪一小步。多轮迭代收敛到平衡态。

### 2.2 刚度 < 1

- 刚度 = 1.0 → 一步到位 → 幂等 → 永远振荡不收敛
- 刚度 < 1.0 → 误差逐轮缩小 → 收敛

### 2.3 多约束竞争

一个顶点被多条约束同时管理。每轮遍历所有约束，各修正一次。同向叠加，反向抵消。**刚度比决定最终平衡点。**

### 2.4 约束执行顺序

```
每帧迭代 N 次:
  ① 形状约束 × 1   (全部顶点 → rest)
  ② 皮筋约束 × M   (M = 皮筋数量, 每条推接触区顶点)
  ③ 体积约束 × 1   (全部顶点 → 均匀外推)
```

皮筋多条是物理事实（确实有 M 条），形状一条也是物理事实（一套菠萝皮）。公平竞争。

## 3. 三条约束

### 3.1 形状约束

```
Δp_i = SHAPE_STIFFNESS × (p_rest[i] − p_current[i])
```

- **基础刚度**：可调，代表菠萝皮的固有弹性
- **线性**：不随位移变化（硬化在皮筋侧实现）

### 3.2 皮筋约束

作用于皮筋高度 ± 影响半径内的顶点。有效目标半径随距离渐变：

```
dy = |顶点Y − 皮筋中心Y|
t = dy / influenceRadius
effectiveTarget = tightR + (当前r − tightR) × smoothstep(0, 1, t)
```

**硬化作用在皮筋侧**：位移越大，皮筋的有效刚度越低。

```
effectiveBandStiffness = BAND_STIFFNESS / exp(HARDENING × displacement × 50)
```

- **基础刚度**：可调，代表皮筋自身的松紧
- **有效刚度**：基础值 ÷ 硬化因子（位移大 → 有效值趋近 0 → 皮筋使不上劲）

物理意义：材料硬化 = 同样的皮筋力产生越来越小的变形 = 等效于皮筋"被削弱"。

### 3.3 体积约束

遍历所有三角形面计算当前体积。若 V_current < V_rest：

所有顶点沿径向（远离 Y 轴）均匀外推，推量 ∝ (VOLUME_STIFFNESS × deficit) / 总表面积。

无 freedom 加权——均匀推，让皮筋区和自由区自然博弈。

## 4. 约束竞争关系

```
皮筋区顶点:
  形状约束往外拉 (线性, 基础刚度)
  vs
  皮筋约束往里推 (基础刚度 / 硬化因子)

  位移小时: 皮筋有效 ≈ 基础刚度 → 正常勒入
  位移大时: 皮筋有效 → 0 → 勒不动了
```

**刚度比决定一切。绝对值被钳在 <1，但比值可以任意大。** 硬化通过在皮筋侧"打折"来拉大比值。

```
自由区顶点:
  形状约束往外拉 vs 体积约束往外推
  → 体积刚度 > 形状刚度 → 两端鼓出
```

无皮筋时只剩形状约束 → 自动弹回原位。

## 5. 参数表

| 参数 | 含义 | 默认值 |
|------|------|--------|
| `SHAPE_STIFFNESS` | 菠萝皮弹性（基础值，线性） | 0.30 |
| `BAND_STIFFNESS` | 皮筋松紧（基础值，被硬化打折） | 0.50 |
| `VOLUME_STIFFNESS` | 体积保持力度 | 0.40 |
| `BAND_INFLUENCE` | 皮筋影响范围（核函数宽度） | 0.60 |
| `HARDENING` | 硬化系数（位移越大皮筋越无力） | 0 |
| `BAND_INITIAL_TIGHTNESS` | 绑上时目标半径 = 当前 × 此值 | 0.80 |
| `PBD_ITERATIONS` | 每帧迭代次数 | 10 |

## 6. 伪代码

```javascript
每帧:
  if bands.length > 0:
    for iter in 1..PBD_ITERATIONS:
      pbdApplyShape(posArr)              // ① 线性拉回 rest
      for band in bands:
        pbdApplyBand(band, posArr)       // ② 基础刚度 / exp(hardening×disp×50)
      pbdApplyVolume(posArr, idxArr)     // ③ 均匀外推补体积
  else:
    for iter in 1..PBD_ITERATIONS:
      pbdApplyShape(posArr)              // 恢复原状

  更新法线、band 视觉、HUD
```
